## Quick Start

Click the button to clone this repository and deploy it on Vercel:

[![](https://vercel.com/button)](https://vercel.com/new/clone?s=https%3A%2F%2Fgithub.com%2FMirasaki-Development%2Fdayz-of-ruin-wiki&showOptionalTeamCreation=false)

Or alternatively, create a build with `npm run build` and host the application with `npm run start`

## Local Development

First, run `npm i` to install the dependencies.

Then, run `npm run dev` to start the development server and visit localhost:3000.
